# Inscripcion - API

Developer must change to "api" directory
```sh
cd api/
```

To install packages:
```sh
$ npm install
```

After, must change db/tools directory from root
```sh
cd db/tools
```

To install database packages:
```sh
$ npm install
```
Then, change your route to the root and apply database patch:
```sh
$ npm run patch
```

To run server:
```sh
$ npm start
```
